# FORSMUSIC

Веб‑плеер в стиле VK Music.

## Демо
- GitHub Pages: https://yadovinartem-jpg.github.io/FORSMUSIC/frontend/

## Структура
- `frontend/` — клиентская часть плеера (HTML/CSS/JS).
- `backend/` — API для загрузки/стриминга через Яндекс.Диск.

## Примечание
Подробные инструкции по запуску и тестированию теперь не дублируются в `README.md` и передаются в чате по запросу.

## Что внутри
- Обновлённый интерфейс плеера в стиле VK Music.
- Фронтенд доступен через GitHub Pages по ссылке выше.

## Вклад в проект
- Pull Request для изменений в этом репозитории направляйте в ветку `main`.
- Для изменений документации используйте именование ветки в формате `feature/readme-update`.
